SPAMDETECTION
Этот проект предназначен для создания базы данных часто встречающихся слов в спам-сообщениях. Базу данных после можно использовать в программах, работающих с текстом для определения заспамленности.
Всё что необходимо сделать - загрузить ваши TXT файлы с предполагаемым спамом в папку resource, скомпилировать learning.cpp, используя команды mkdir build && cd build && cmake ../ && make в корневой директории, или используя g++ (в этом случае необходимо в коде learinig.cpp убрать "../" во всех функциях).

Результатом работы программы являются два TXT файла. selection.txt показывает общую информацию об изначальных файлах, model.txt является файлом с MySQL командами, для быстрого создания готовой базы данных.
